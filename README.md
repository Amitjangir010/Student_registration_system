# Student_registration_system

The Student Registration App is a web-based application built using Flask framework to manage student information in a school or educational institution. It provides functionality for adding, updating, and deleting student data in a user-friendly manner. The app also includes a login system to ensure data security and privacy.

required_packages = [

## Files

The project contains the following files:

1. `main.py` - The main Python file containing the Flask application code.
2. `dashboard.html` - HTML template for the dashboard page.
3. `index.html` - HTML template for the home or landing page.
4. `login.html` - HTML template for the login page.
5. `update_student.html` - HTML template for updating student information.
6. `education.png` - Sample image for educational purposes (can be replaced).
7. `password.png` - Sample image for password-related visuals (can be replaced).
8. `style.css` - CSS file for custom styling of the web application.
9. `styles.css` - Additional CSS file for enhancing the app's visual appeal.
10. `th.jpg` - Thumbnail image used in this README file (replace with your own).
11. `user.png` - Sample image for user profile pictures (can be replaced).

## Installation

1. Make sure you have Python and Flask installed on your system.
2. Clone this repository to your local machine.
3. pip install required packages.
5. Start the application by running `python main.py` in the terminal.
6. Visit `http://localhost:5000` in your web browser to access the app.
7. use database name as "root" , username="abcd" , password="1234" (its default , you can change it in main.py code )

## Usage

1. Upon launching the app, you'll see the login page.
2. If you don't have an account, click on the "Register" link to create one.
3. After logging in, you'll be redirected to the dashboard where you can manage student records.
4. Use the navigation menu to add, update, or delete student information.
5. You can also upload student profile pictures for a more personalized experience.
